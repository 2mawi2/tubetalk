1. Install TubeTalk: Tambahkan ekstensi TubeTalk ke browser Chrome-mu dari Chrome Web Store.

Pertanyaan yang Sering Diajukan (FAQ)
----------------------------------------

PERTANYAAN: Apa yang bikin TubeTalk beda dari ekstensi peringkas YouTube lainnya?
JAWABAN: TubeTalk nggak cuma kasih ringkasan video instan, tapi juga punya fitur chat AI interaktif dalam video YouTube. Plus, kamu bisa pakai API key OpenAI atau OpenRouter sendiri, yang bikin data dan privasi kamu terjamin.

PERTANYAAN: Gimana cara dapetin API key OpenAI atau OpenRouter?
JAWABAN: Kamu bisa daftar untuk dapetin API key di website OpenAI (https://openai.com/) atau website OpenRouter (https://openrouter.ai/). Setelah dapat API key, tinggal masukin ke pengaturan TubeTalk.

PERTANYAAN: Apa TubeTalk bisa dipake di semua video YouTube?
JAWABAN: TubeTalk bisa dipake di video YouTube yang punya caption. AI-nya pake caption ini buat bikin ringkasan dan jawaban yang akurat.

PERTANYAAN: Apa TubeTalk gratis?
JAWABAN: Ya, install dan pake TubeTalk gratis. Tapi, pemakaian API key OpenAI atau OpenRouter mungkin ada biayanya tergantung paket yang kamu pilih di layanan tersebut.

PERTANYAAN: Apa TubeTalk support banyak bahasa?
JAWABAN: Tentu! TubeTalk bisa dipake dengan caption YouTube dalam berbagai bahasa, jadi kamu bisa interaksi dengan video dalam bahasa yang kamu suka.

PERTANYAAN: Seberapa aman API key-ku di TubeTalk?
JAWABAN: API key kamu disimpan lokal di browser dan nggak pernah dikirim ke server kami. Semua interaksi langsung antara browser kamu dan OpenAI atau OpenRouter.

PERTANYAAN: Bisa nggak kustomisasi model AI yang dipake TubeTalk?
JAWABAN: Bisa, karena kamu pake API key sendiri, kamu bisa atur model OpenAI mana yang mau dipake sesuai akses API-mu.

PERTANYAAN: Gimana kalo videonya nggak ada caption?
JAWABAN: TubeTalk butuh caption buat bisa jalan dengan baik. Kalo video nggak punya caption, AI-nya nggak bisa memproses kontennya.

PERTANYAAN: Gimana cara update atau ganti API key?
JAWABAN: Buka pengaturan TubeTalk dengan klik ikon ekstensi di toolbar browser. Di situ kamu bisa update atau ganti API key kapan aja.

PERTANYAAN: Butuh bantuan atau mau kasih feedback?
JAWABAN: Untuk dukungan atau feedback, silakan buka Issue di repositori GitHub kami: https://github.com/2mawi2/tubetalk/issues
