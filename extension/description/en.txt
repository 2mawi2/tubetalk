Getting Started:

1. Install TubeTalk: Add the TubeTalk extension to your Chrome browser from the Chrome Web Store.
2. Enter Your API Key: Open the extension settings and input your OpenAI or OpenRouter API key.
3. Open a YouTube Video: Navigate to any YouTube video that has captions available.
4. Activate TubeTalk: Press Alt+Y to start interacting with the video using AI chat and summaries.

Privacy Focused:

When using your own API key, all AI processing happens through your direct connection with OpenAI or OpenRouter. Your data stays secure, and you have full control over your API usage.

Note:

- YouTube Captions Required: TubeTalk requires YouTube video captions to function effectively.

Frequently Asked Questions (FAQ)
----------------------------------------

QUESTION: What makes TubeTalk different from other YouTube summarizer extensions?
ANSWER: TubeTalk not only provides instant video summaries but also offers interactive AI chat within YouTube videos. Plus, you can use your own OpenAI or OpenRouter API key, ensuring full control over your data and privacy.

QUESTION: How do I get an OpenAI or OpenRouter API key?
ANSWER: You can sign up for an API key on the OpenAI website (https://openai.com/) or OpenRouter website (https://openrouter.ai/). Once you have your API key, enter it into TubeTalk's settings.

QUESTION: Can I use TubeTalk with any YouTube video?
ANSWER: TubeTalk works with YouTube videos that have captions. The AI uses these captions to generate accurate summaries and answers.

QUESTION: Is TubeTalk free to use?
ANSWER: Yes, TubeTalk is free to install and use. However, using your OpenAI or OpenRouter API key may incur charges based on your usage plan with those services.

QUESTION: Does TubeTalk support multiple languages?
ANSWER: Absolutely! TubeTalk works with YouTube captions in multiple languages, allowing you to interact with videos in your preferred language.

QUESTION: How secure is my API key with TubeTalk?
ANSWER: Your API key is stored locally in your browser and is never sent to our servers. All interactions are directly between your browser and OpenAI or OpenRouter.

QUESTION: Can I customize the AI model used by TubeTalk?
ANSWER: Yes, since you're using your own API key, you can configure which OpenAI models you want to use based on your API access.

QUESTION: What if a video doesn't have captions?
ANSWER: TubeTalk requires captions to function properly. If a video doesn't have captions, the AI won't be able to process the content.

QUESTION: How do I update or change my API key?
ANSWER: Go to TubeTalk's settings by clicking on the extension icon in your browser toolbar. From there, you can update or change your API key at any time.

QUESTION: Need help or want to give feedback?
ANSWER: For support or feedback, please open an issue on our GitHub repository: https://github.com/2mawi2/tubetalk/issues
